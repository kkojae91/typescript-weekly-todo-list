# TypeScript Weekly Todolist Project

## 프로젝트 기획

- 서비스 이용자들의 주간 계획을 보다 효과적으로 관리할 수 있는 프로그램 만들기
- 참고 사이트 : notion

## 핵심 기능 구현

- Drag and drop 활용 (요소 이동)
- Storage 활용해 사용자의 기록 저장후 재접속시 기록 유지
- Toggle을 통해 dark 모드 구현
- 반응형 구현(mobile, desktop)
- dataset 활용
